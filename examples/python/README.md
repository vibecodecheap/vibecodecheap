# Python Examples

Examples using OpenAI SDK with VibeCodeCheap API.

## Setup

```bash
pip install openai
```

## Examples

### basic.py
Simple chat completion example.

```bash
python basic.py
```

### streaming.py
Streaming response example.

```bash
python streaming.py
```

## Configuration

Set your API key:

```bash
export OPENAI_API_KEY=your_api_key
```

Or set it directly in the code (not recommended for production).
