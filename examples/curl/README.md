# cURL Examples

Examples using cURL with VibeCodeCheap API.

## Examples

### basic.sh
Simple chat completion example.

```bash
chmod +x basic.sh
./basic.sh
```

### streaming.sh
Streaming response example.

```bash
chmod +x streaming.sh
./streaming.sh
```

## Configuration

Replace `your_api_key` in the scripts with your actual API key.

Or use environment variable:

```bash
export API_KEY=your_api_key
```
